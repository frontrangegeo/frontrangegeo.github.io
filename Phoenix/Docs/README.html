Phoenix Docs README

Building

To compile .md to .html:

./compile input.html

Then go to that file path in your web browser. If it’s already open in your browser, just refresh the webpage after recompiling.

Organization

We’ll place documentation .md files and resources in this (phoenix_clean/Docs/) directory. Please put resources like images/videos in the Docs/Resources path, and link to them appropriately.

Samples

SamplePage.md shows the basic page structure and how to do various things with Markdown.

index.md will be the main page, and will link to the other pages. We’ll fill in the table of contents there as we go.

Resources/PhoenixGuide.pdf is a previously made pdf showing some basic processing in phoenix.
